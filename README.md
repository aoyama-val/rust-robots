# rust-rectangle-eraser

A realtime robots game written in Rust.




## Requirement

- SDL 2
- Rust


## Controls

```
h      : Move left
j      : Move down
k      : Move up
l      : Move right
y      : Move up and left
u      : Move up and right
b      : Move down and left
n      : Move down and right
t      : Teleport to a random location
Enter  : Restart
```
